Leave

Applied leave can be cancelled at any time before approval or after approval by the Student. However leave cannot be cancelled after leave started.

One leave request / Late hour permission only permitted at any given point of time. i.e System will prevent leave requests if any leave is found to be waiting for approval or any leave approved but not closed
